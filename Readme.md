# 🎮 Tic Tac Toe – Play & Enjoy!

Welcome to **TicTactics.netlify.app** – a fun, interactive, and beautifully designed Tic Tac Toe game built with pure HTML, CSS, and JavaScript! ✨

---

## 🕹️ Features

- **Classic 3x3 Grid:** Play the timeless game you know and love.
- **Responsive Design:** Looks great on desktop and mobile devices.
- **Instant Feedback:** See the winner or draw instantly after each game.
- **Reset Button:** Start a new game anytime with a single click.
- **Celebratory Animation:** Enjoy a congratulatory GIF when you win! 🎉
- **Sound Effects:** Immersive audio for moves, wins, and draws. 🔊

---

## 🚀 How to Play

1. **Open the game** in your browser.
2. **Click** on any empty box to make your move.
3. **Alternate turns** between "X" and "O".
4. **Win** by getting three in a row, column, or diagonal.
5. **Draw?** No problem! Hit the **Reset** button to play again.
